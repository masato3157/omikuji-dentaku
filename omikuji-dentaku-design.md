# 「おみくじ電卓」設計概要書


## アプリ概要

普通に使える電卓アプリ。
計算結果は正確に出る。
ただし、計算結果と一緒に「おみくじ」が表示される。
計算結果とおみくじには何の関係もない。


## アプリ名

**おみくじ電卓**

ハッシュタグ：#おみくじ電卓


## フォーク元

このプロジェクトは「いい加減な電卓 (iikagen-dentaku)」v0.7.0 からフォークしました。
「いい加減な電卓」の設計思想やUIをベースに、おみくじ機能を強化しています。


## 基本機能

1. 普通の電卓として四則演算ができる
2. 計算結果を出すたびに「おみくじ」がチラ見せで表示される
3. 「続きを見る」をタップすると広告が表示される
4. 広告の後、おみくじの「続き」が読める


## マネタイズ

広告収益モデル。
「続きが気になる」という心理で広告視聴のハードルを下げる。


## 「おみくじ」の設計思想

### 方向性

- 吉凶を占う形式ではない
- 意味深で、読んだ瞬間「何？」と気になる一言
- 良いとも悪いとも言っていないのに、勝手に心当たりを探してしまう感じ


### トーンの振り幅（飽きさせない工夫）

- 意味深なやつ
- 完全に脱力系
- たまにちょっと優しい
- たまに毒がある
- 計算に絡むときと、全く関係ないとき

「次、どっちが来るかわからない」という期待感を作る。


### 構造案：おみくじ方式（推奨）

カテゴリごとにパターンを作り、ランダムに組み合わせて表示する。

カテゴリ例：
- 今日のひとこと
- 気をつけること
- ラッキーナンバー
- 待ち人
- 失せ物
- あの件

6カテゴリ×10パターン＝60個作るだけで、組み合わせは100万通りになる。

出力例：
```
今日のひとこと：知ってました
気をつけること：左側
ラッキーナンバー：たぶん7
待ち人：来るけど遅い
失せ物：冷蔵庫の奥
あの件：まだ間に合います
```

全部いい加減だけど、妙に気になる。


### チラ見せと続きの設計

チラ見せは短い一言で引っ掛ける。
続きはある程度ボリュームを持たせて満足感を出す。
おみくじのように複数の項目が並んでいると、そのうちの一行がピンときたりする。


## 隠し要素（イースターエッグ）

### 計算結果に連動するもの

- 数千〜1万円 ÷ 2〜6 →「割り勘ですか？」
- ÷12 →「月割りですね」
- ×1.1 や ×1.08 →「税込み気にするタイプですね」
- 結果が0 →「...ですよね」
- 結果がマイナス →「知ってました」
- ゾロ目 →「揃いましたね。だから何ってわけじゃないですが」


### 特殊な操作に連動するもの

- 0で割る → 電卓がバラバラに吹っ飛ぶ演出、「だから言ったのに」と表示
- C（クリア）連打 →「落ち着いて」
- 何も入力せず= →「何がしたいんですか？」
- ずっと放置 →「...まだいます？」
- 1+1= →「合ってます。不安だったんですか？」


### 隠しコマンド

特定の順番でキーを押すと何かが起きる。

- 「573」（コナミ）で特別な演出
- 「4649」でヤンキーっぽい音
- 「0840」（おはよー）で朝の目覚まし音
- AC長押しでシークレットモード突入


### 音の演出

- 普段のキー音：ピコピコ可愛い感じ
- =を押したとき：ちょっと荘厳な「ゴーン」
- 0で割る：爆発音
- マイナス結果：「チーン」


## プラットフォーム

### 推奨：LINEミニアプリ（LIFF）

理由：
- インストール不要（心理的ハードル低い）
- LINEで共有が簡単（「なにこれ笑」→そのままトークで送れる）
- 結局Webアプリなので、ブラウザからも普通に使える
- 同じコードでLINE内とブラウザ両対応
- App Storeの審査がない
- 広告はGoogle AdSenseなど普通に貼れる


## 拡散戦略

### 最初のプラットフォーム：X（Twitter）一択

理由：
- スクショ1枚で完結する面白さ
- 「125×8=1000」「合ってます?」→説明なしで「何？」ってなる
- TikTok/YouTubeは動画にする意味が薄い（電卓ポチポチ押す絵面が地味）
- Instagramはおしゃれ感ゼロで世界観が合わない


### X戦略

毎日1〜2回、違う計算結果とおみくじの組み合わせを投稿する。
淡々と。説明しない。リンクも最初は貼らない。
「何これ？」を溜めていく。

投稿例：
```
48÷6=8
「知ってました」
#おみくじ電卓
```

「気になる人はプロフにリンクあります」で追わせる。


### SNSシェア機能

アプリ内に「ポストする」ボタンを設置。
計算結果とおみくじがセットでシェアされる。

シェア時のフォーマット：
```
125×8=1000
「合ってます?」
#おみくじ電卓
```


### コンテンツ化

- 「バカバカしい電卓アプリ作ってみた」でブログ記事
- 開発過程をYouTubeにする
- 「おみくじ」の文章を考える過程自体をコンテンツにする

アプリ単体で勝負するのではなく、コンテンツの一部としてアプリがある構造。


### AI検索対策（これからのSEO）

AIが参照したくなる状態を作る。
複数のソースで言及されていることが重要。

- 個人ブログで紹介記事
- Xで複数人が言及
- アプリ紹介サイトに載る
- YouTubeで誰かがレビュー

「おみくじ電卓」で検索したら色んなところで出てくる状態が理想。


## 初期ユーザー獲得

最初の100人は自力で集める。
知り合い、SNSのフォロワー、既存のブログ読者に「ちょっと使ってみて」を地道にやる。

バズるかどうかは運もある。
作るコスト・維持コストは低いので、出してみて反応を見るのが一番早い。
焦らず種を蒔いておく。
