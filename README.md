# おみくじ電卓 (omikuji-dentaku)

普通に使える電卓アプリ。計算結果は正確に出る。
ただし、計算結果と一緒に「おみくじ」が表示される。

## 技術スタック

- **フレームワーク**: Vite + React
- **スタイリング**: Tailwind CSS
- **状態管理**: useState / useEffect

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```

## フォーク元

このプロジェクトは「いい加減な電卓 (iikagen-dentaku)」v0.7.0 からフォークしました。
